1. Run command to install venv:
	Windows: python -m venv venv
	linux+Mac: python3 -m venv venv
2. Active venv:
	Windows: venv\Scripts\activate
	linux+Mac: source venv/bin/activate
3. In venv run:
	pip install -r requirements.txt
	pip install yt-dlp
	pip install PyNaCl
